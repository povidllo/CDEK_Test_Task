# CDEK_Test_Task
Тестовое задание для CDEK
Для запуска нужно:
```sh
cd frontend
npm install
npm run dev
```
На главном экране разместил все 3 основные версии. 
(в качестве тегов брал значения из поля *rubrics*, не уверен, что это прям они, но если нет, то переделать за несколько минут можно)
Использовал typescript для разработки.
Основной компонент это ```NewsComponent```, принимает следующие пропсы:
```typescript
  variant?: "business" | "company";
  perPage?: number;
  empty?: boolean;
```
Первое тип новостного блока, второе сколько элементов на странице, и empty для теста пустого блока
Добавил skeleton loading состояние во время загрузки
Для стилизации использовал tailwind, tabler icons
Для взаимодействия с api использовал tanstack query
